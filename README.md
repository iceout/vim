我的vimrc
=========

简介
----
个人vimrc存档

cycle
-----
`<Leader>a` true,false 等等循环
`:h cycle`

bufexplorer 
-----------
`<Leader>be` " 打开buf
`<Leader>bs` " 水平分割窗口
`<Leader>bv` " 垂直分割窗口

tasklist  
--------
`<Leader>t` " 查看tasklist

NERD\_commenter 
--------------
`<Leader>c<space>` " 根据第一行的状态来切换所选行的注释状态
`<Leader>cm` " 块注释
`<Leader>cc` " 行注释

EasyMotion
--
`<Leader><Leader>w` 显示序号,然后按序号跳转
